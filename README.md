# 📘 Hora de Codar 4

Este repositório reúne os exercícios do **Hora de Codar 4**, dando continuidade ao desenvolvimento da lógica de programação com **JavaScript**, com foco ainda maior no uso de estruturas de repetição.

## 🚀 Objetivo

O objetivo principal desta etapa foi reforçar o uso de:

* `while`
* `do...while`
* `for`

com mais controle sobre fluxos de repetição e validações, tornando o código mais eficiente e organizado.

## 🧠 Conteúdos abordados

De forma geral, os exercícios trabalham com:

* Repetições controladas por condição (`while`)
* Execução garantida ao menos uma vez (`do...while`)
* Contagens e iterações com `for`
* Validação de entradas do usuário
* Acúmulo de valores e cálculos

O destaque ficou no uso do **`while`**, sendo bastante utilizado para:

* Validar dados até que estejam corretos
* Criar loops baseados em condições dinâmicas
* Controlar execuções sem número fixo de repetições

## 🛠️ Tecnologias utilizadas

* JavaScript (puro)
* Execução via terminal ou navegador

## 📌 Considerações finais

O **Hora de Codar 4** aprofunda ainda mais a prática com estruturas de repetição, principalmente o `while`, ajudando a desenvolver um pensamento mais lógico e estruturado para resolver problemas. Foi essencial para ganhar mais segurança na criação de algoritmos.
